# kms_server
kms server

# depend on vlmcsd
https://github.com/Wind4/vlmcsd

# install
```shell
git clone https://github.com/easonjim/kms_server.git
cd kms_server
bash install_kms_server.sh
```

# uninstall
```shell
bash uninstall_kms_server.sh
```

# vlmcsd service command
```shell
# start
service vlmcsd start
# stop
service vlmcsd stop
# status
service vlmcsd status
```
